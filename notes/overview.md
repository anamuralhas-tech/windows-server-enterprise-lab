# Overview

This repository documents a Windows Server 2025 enterprise lab focused on the design and implementation of a small business infrastructure.

The project includes:
- static network configuration
- Active Directory Domain Services
- DNS
- organizational units, security groups, and domain users
- shared folders and NTFS permissions
- scheduled and manual backups
- restore validation
- Windows 11 client domain join
- controlled Remote Desktop access

The lab was documented as a professional technical project, combining implementation evidence with business-oriented justification and validation.
